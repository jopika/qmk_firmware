# Custom keymapping for Tada68

This is a layout that is similar to a normal QUERTY layout, adding some functionality 

```
Layer 1 (Base layer)
   ,----------------------------------------------------------------.
   |Esc | 1|  2|  3|  4|  5|  6|  7|  8|  9|  0|  -|  =|Backsp |~ ` |
   |----------------------------------------------------------------|
   |Tab  |  Q|  W|  E|  R|  T|  Y|  U|  I|  O|  P|  [|  ]|  \  |Del |
   |----------------------------------------------------------------|
   |FN   |  A|  S|  D|  F|  G|  H|  J|  K|  L|  ;|  '|Return   |PgUp|
   |----------------------------------------------------------------|
   |Shift   |  Z|  X|  C|  V|  B|  N|  M|  ,|  .|  /|Shift | Up|PgDn|
   |----------------------------------------------------------------|
   |Ctrl|Win |Alt |        Space          |Alt| FN|Ctrl|Lef|Dow|Rig |
   `----------------------------------------------------------------'

Layer 2 (Upper Layer)
   ,----------------------------------------------------------------.
   |~ `| F1|F2 |F3 |F4 |F5 |F6 |F7 |F8 |F9 |F10|F11|F12|Del    |Ins |
   |----------------------------------------------------------------|
   |     |   |   |   |   |   |   |   |   |   |PTS|   |   |     |Hme | // PTS = Print Screen
   |----------------------------------------------------------------|
   |         |    |   |=M=|   |Lef|Up |Dow|Rig|   |   |        |End |
   |----------------------------------------------------------------|
   |        |MUT|VL-|VL+|   |   |   |Bl-|BL |BL+|   |      |.M.|    | // Arrow keys-> Stop
   |----------------------------------------------------------------| // Previous, Play/Pause, Next
   |    |CAPS|    |                       |   |   |    |<<M|=M=|M>> |
   `----------------------------------------------------------------'
   MUT == Mute
   VL- == Volume Control: Decrease
   VL+ == Volume Control: Increase
   Bl- == Backlight (previous)
   BL  == Backlight (toggle)
   BL+ == Backlight (next)
   PTS == Print Screen
   .M. == Media Stop
   =M= == Media Play/Pause
   <<M == Media Previous
   M>> == Media Next
```
